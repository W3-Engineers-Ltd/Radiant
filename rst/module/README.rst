Introduction to modules
=======================

Radiant is evolving to a “LEGO model” which means becoming a modular
system. Each module is one of the building blocks and you can build the
application you want with these blocks. Since ``Radiant 1.0`` these
modules are listed below:

-  `Session Module <./session.html>`__
-  `Cache Module <./cache.html>`__
-  `Logs Module <./logs.html>`__
-  `Httplib Module <./httplib.html>`__
-  `Context Module <./context.html>`__
-  `Task Module <./task.html>`__
-  `Admin Module <admin.html>`__
-  `Config Module <./config.html>`__

There are also these independent modules, which are available in
separate repositories:

-  `i18n Module <./i18n.html>`__
